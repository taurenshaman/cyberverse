### Usage
* Request full screen mode: `#`, `F`
* Call a **Info Panel**: `!`, `I`
* Node. View a specific angle using the parameter: `num` or `n`.

### Deployment
All are static files. No database required.  
Just upload files in `src` directly and browse.
